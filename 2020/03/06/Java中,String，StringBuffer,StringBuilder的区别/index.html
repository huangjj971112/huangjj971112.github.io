StringBuilder和StringBuffer的共同点:
  StringBuffer和StringBuilderd的父类都是AbstractStringBuilder(抽象类)，StringBuffer和StringBuilder都可以调用AbstractStringBuilder的方法.append(),.insert().
  抽象类与接口的其中一个区别是：抽象类中可以定义一些子类的公共方法，子类只需要增加新的功能，不需要重复写已经存在的方法；而接口中只是对方法的申明和常量的定义。
