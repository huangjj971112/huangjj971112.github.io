查询数据

  查询数据表中所有数据:
select * from [table-name]
  查询某列数据:
select [column-name],[column-nname] from [table-name];
  条件查询：
select [column-name],[column-name] from [table-name] where [queryCondition]
  egs:select id,name from test01 where id=1;

update更新数据表

update [table-name] set filed01=newvalue, filed02=newvalue where [update-condition];
  egs:
update test01 set name="hello" where id=1;

删除数据

delete from [table-name] where [condition];\\如果没有where条件，整个数据表都会被删除

like语句

  在mysql数据表的查询中，有时候我们需要查寻某列的属性值中包含有某个字符串的数据，这时候like就可以派上用场了，在like中“%”表示任意字符。如果在like语句中没有使用“%”，就等同于“=”了。下面是like语句的使用:
select filed01,filed02 from [table-name] where filed02 like %c;
  egs:
select id from test01 where name like '%c%';//查询test01表中name中包含c的id

排序

  mysql中的排序使用order by,其中asc代表增序，desc代表降序；默认增序排列。
select field1, field2, from [table-name] order by field1 [asc [desc];
  egs:
select id,name from test01 order by id desc;

分组

select filed01,filed02 from [table-name] group by filed01;
  egs:
select id,name ,count(name) from test01 group by name;//count()函数用于计算同一个name属性值出现了几次

查看数据表字段

show columns from [table-name];
