Limit

  selct * from tableName limit rowNumber用于获取表的前rowNumber行数据。

Like

  select * from tableName where colunmName like %C获取数据库表中列名以C开头的所有数据。如果是C%表示以C结尾，%C%表示包含C。

IN

in操作符允许在where中归定多个值。
  selecr * from tableName where columnName in (value1,value2);表示获取列的值为value1,value2的数据。

Between

  between value1 and value2用于获取处于俩个值之间的数据,包括value1和value2。
  not between and 表示值不存在于俩个值之间的数据。

别名

  使用别名的情况:1.查询过程中超过一个表；2.查询中列的名很长或可读性很差；3.使用函数；4.将多个列合在一起。
  列别名：select name as n from tablename
  表别名：select t.id,t.name ,t1.class from tableName as t,tableName1 as t1 where t.id=t1.id;

join

join用于多表查询，其中常用的查询有inner join即韦恩图的交集，left join、right join部分如果想获取俩个表的并集可以使用union将俩个表的left join和right join连接起来即可。关于left join和rigth join的区别，请见这里。

||连接符

  ||连接符用于拼接字符串，例如"aa"||"bb"等价于"aabb",其中contact只能连接俩个字符串，而||可以连接多个字符串。

date_formate()

  date_formate()用于将日期格式转换成字符串格式，例如date_formate(2020-05-04,"Y%-m%-d%")。

str_to_date()

  str_to_date，用于将字符串转为日期格式。例如(str_to_date('2018-05-06','%Y-%m-%d'))。

select 1 from tablename

  

insert的几种方式

  value、values、select三种，其中insert into tableName (index1) select index0 from table0使用select插入不需要使用values或value。相比之下，插入多行数据的时候使用value，效率高，values适用于单行插入。

SQL通配符

  mysql中的通配符通常用于替代字符串中的任意字符。mysql常用的通配符如下：

通配符 描述
% 替代0到多个字符
_ 替代一个字符
[charlist] 字符列中的任一单字符
[^charlist]或[!charlist] 不在字符列中的任一单字符

  seclect * from tableName where columnName like '_C_';表是从数据库中读取某列的值的发一个字符是以任一字符开始，第二个字符是C，最后一个字符是以任一字符结束的行的值。
  mysql中使用regexp和not regexp来操作正则表达式。
  select * from tableName where columnName regexp '^[ABc]'表示选取某列以A,B,c开头的所有数据。
  select * from tableName where columnName regexp '^[A-M]'表示获取以A到M开头的所有数据。
  select * from tableNaame where columnName regexp'^[^ABC]'表示获取不以A,B或C开头的所有数据。
